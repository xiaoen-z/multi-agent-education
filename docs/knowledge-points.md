# 知识点体系（200+ 知识点示例）

> 本文档展示系统内置的知识图谱结构，以初中数学为例

## 知识图谱结构

知识图谱是一个 **有向无环图 (DAG)**，箭头表示前置依赖关系。

```
四则运算 ─┬─→ 分数运算 ──→ 概率初步
          │              └─→ 数据统计
          ├─→ 负数 ─────→ 代数式 ──┬─→ 一元一次方程 ──┬─→ 二元一次方程组
          │              │         │                  ├─→ 不等式
          │              │         └─→ 因式分解 ──────┴─→ 一元二次方程 ──→ 二次函数
          │              │         └─→ 数列
          │              └─→ 平面直角坐标系 ──→ 一次函数
          │
          ├─→ 勾股定理 ──→ 相似三角形
          │              └─→ 三角函数基础
          │
          └─→ 集合 ──→ 简易逻辑
```

## 扩展方向

实际部署时可扩展到200+知识点，覆盖：
- 初中数学完整体系（约80个知识点）
- 高中数学基础（约120个知识点）
- 也可扩展到物理、化学等学科

知识点数据存储在 `knowledge_graph.py` 中的 `build_sample_math_graph()` 函数。
